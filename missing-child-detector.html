Missing Child Detector

A GPS and GSM-enabled IoT device designed to enhance child safety and provide real-time location tracking for parents.

Back to Portfolio

Project Overview

The Missing Child Detector is an innovative IoT solution developed as a team project to address child safety concerns. This device combines GPS and GSM technologies to provide real-time location tracking and emergency alerts.

Parents can monitor their child's location through a mobile application and receive instant notifications if the child moves beyond predefined safe zones. The system also features an emergency button that the child can press to immediately alert parents and authorities.

Key Features

  • Real-time GPS location tracking
  • GSM connectivity for remote communication
  • Geofencing with customizable safe zones
  • Emergency alert button
  • Low battery alerts
  • Mobile app integration for parents
  • Compact and child-friendly design

Technical Implementation

The device was built using Arduino microcontroller, GPS module for location tracking, GSM module for communication, and various sensors. The system integrates with a mobile application that allows parents to monitor their child's location in real-time.

We implemented geofencing technology that triggers alerts when the child exits predefined safe areas. The emergency button sends immediate notifications with the exact GPS coordinates to registered contacts.
