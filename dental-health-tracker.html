Final Year Project

Dental Health Tracker

A mobile application using gamification and reminder systems to improve children's dental health through engaging features and parental involvement.

Back to Portfolio

Project Overview

This research examines the development and implementation of modern technology to redefine the way dental health is taught and maintained. The application explores the efficiency of features such as gamified routine tracking, dental checkup reminders, and educational resources in fostering active learning, behavioural consistency, and improved dental hygiene practices.

The app's design emphasizes parental involvement and child engagement through comprehensive tools that encourage both routine adherence and proactive dental care. The application's architecture comprises multiple components, including routine task management, notification systems for reminders, content delivery for video-based education, user interaction tracking, database management, and parental dashboards.

Key Features

  • Gamified routine tracking with badges and rewards
  • Personalized dental checkup reminders
  • Educational video library for dental health
  • Progress monitoring and streak tracking
  • Parental dashboard for monitoring children's progress
  • Interactive task completion system
  • Firebase Realtime Database integration

Technical Implementation

The application was developed using Flutter framework with Firebase as the backend database. The system integrates real-time data synchronization, push notifications for reminders, and a comprehensive user management system.

The gamified approach incentivizes consistency through badges and rewards for daily and weekly streaks. Interactive features, like personalized reminders and progress monitoring, enable parents to guide their children effectively. By aligning educational content with practical tasks, the app bridges the gap between knowledge and action, encouraging better long-term dental hygiene habits and enhancing parental awareness.
